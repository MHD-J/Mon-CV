📖 Contexte

Projet personnel : développement d'un endless runner 2D en C++ avec SFML. Le joueur court automatiquement et doit éviter des obstacles.

⚙️ Fonctionnalités

  • Collisions précises (hitboxes réduites)
  • Difficulté progressive (vitesse + fréquence)
  • Système de vies (3 cœurs)
  • Animations (course, saut, glissade, mort)
  • Sauvegarde du meilleur score
  • Menu interactif et pause

💻 Extrait de code (Collisions)

bool checkCollision(const Sprite& a, const Sprite& b, float tolerance) {
FloatRect rectA = a.getGlobalBounds();
rectA.left += tolerance;
rectA.width -= 2 * tolerance;
return rectA.intersects(b.getGlobalBounds());
}
